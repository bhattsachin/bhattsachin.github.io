--- layout: post title: "Regex grouping in R" date: 2015-03-09 23:35:10 --- Problem statement: Extract number of bed, bath and area from the follwing string.

"3 bd, 2 ba, 1,300 sqft"

We need to create a group of 3 regex statements to match this pattern. {% highlight R %} # using str_match library(stringr) base<-"3 bd, 2 ba, 1,300 sqft" str_match(base, "([0-9.]+) bd, ([0-9.]+) ba, ([0-9,]+) sqft") {% endhighlight %} Results:
[,1] [,2] [,3] [,4]
[1,] "3 bd, 2 ba, 1,300 sqft" "3" "2" "1,300"
