Test regular expressions on web.

http://www.fileformat.info/tool/regex.htm

One example for Pattern Matching in Java
Input String is : (ROOT (NP [37.472] (NP [8.500] (JJS [2.532] best)) (NP [9.302] (NN [6.697] drug)) (NP [15.804] (JJ [10.291] antibiotic))))

We have to find patterns like (word [float] word) in above input.

Expected String output:

(JJS [2.532] best)

(NN [6.697] drug)

(JJ [10.291] antibiotic)

Step 1. Create a regular expression using above web url (so that you keep on testing it on fly)

String regex = "\\([a-zA-Z0-9]+ \\[[0-9]+\\.[0-9]+\\] [a-zA-Z0-9]+\\)";

String input = "(ROOT (NP [37.472] (NP [8.500] (JJS [2.532] best)) (NP [9.302] (NN [6.697] drug)) (NP [15.804] (JJ [10.291] antibiotic))))";

Step 2. Compile regular expression using Pattern Class

Pattern pattern = Pattern.compile(regex);

Step 3. Apply this regular expression on the input

Matcher matcher = pattern.matcher(input);

Step 4. Print this output on console
while (matcher.find()) {
System.out.println(input.subSequence(matcher.start(), matcher.end()));
}

That is all to it :)
